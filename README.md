# API Login - Proyecto Python con Flask y bcrypt

Este proyecto es una **API de login y registro de usuarios** desarrollada en Python usando Flask y SQLite, con contraseñas encriptadas usando bcrypt.  

Sirve como ejemplo de buenas prácticas de backend y manejo de contraseñas de forma segura.

---

## 🔹 Funcionalidades

- Registro de usuarios (`/register`)  
- Login de usuarios (`/login`)  
- Contraseñas guardadas en formato seguro (bytes) usando bcrypt  
- Mini script de prueba incluido (`test_api.py`)  

---

- Python 3.10 o superior
